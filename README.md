# Project Overview
This project was created as part of the Deployment of Static Websites practical task in the Stellenbosch University Full Stack Web & Software Engineer Bootcamp. The objective was to build and deploy a personal CV webpage using HTML and CSS, and publish it using GitHub Pages. Rather than creating a traditional CV layout, I designed the interface to resemble a developer code editor environment. This approach visually communicates my transition from digital media design into software development.

# Technologies Used
HTML
CSS
Bootstrap
Google Fonts
GitHub Pages for deployment
